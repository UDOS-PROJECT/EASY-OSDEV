Useful Hercules commands

// Enable tracing for DASD
t+1b9

// Enable instruction tracing
t+

// Boot the DASD
ipl 1b9

Notice that Hercules has a DIAG 8 call to print debug statments, you can use it to issue commands to the Hercules console
